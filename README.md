# Homebrew Tap for MCP Hub

Homebrew formulae for [MCP Hub Platform](https://mcp-hub.info) tools.

## Install

```bash
brew install mcp-hub-corp/tap/mcp
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `mcp` | Run MCP servers without blindly trusting them |
